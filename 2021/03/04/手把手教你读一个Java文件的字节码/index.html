小结

Java 的字节码还是要比汇编简单一些。

这里再提一点，当要初始化一个 int 时（在 JVM 中：bool，byte，char，short 都是 int），根据不同的数字所占的位数不同，分别需要如下几个命令，方括号中给出了命令适用的范围

  • iconst: [-1, 5]
  • bipush: [-128, 127]
  • sipush: [-32768, 32767]
  • idc: any int value
